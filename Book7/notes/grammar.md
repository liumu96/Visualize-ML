## Scipy

| API                        | DESCRIPTION                                  |
| -------------------------- | -------------------------------------------- |
| `scipy.stats.kurtosis()`   | 计算峰度                                     |
| `scipy.stats.normaltest()` | Omnibus 正态检验                             |
| `scipy.stats.skew()`       | 计算偏度                                     |
| `scipy.stats.t.ppf()`      | 求解 t 分布的逆累积分布函数                  |
| `scipy.stats.t.sf()`       | 求解 t 分布的互补累积分布函数 CCDF = 1 - CDF |
